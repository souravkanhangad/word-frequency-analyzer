# Word Frequency Analyzer

A simple Java program that reads a text file and calculates how many times each word appears.

## Features

- Reads text from a file
- Converts text to lowercase
- Removes punctuation
- Counts word frequencies using HashMap

## Example Input

Java is a powerful language. Java is widely used.

## Example Output

java : 2  
is : 2  
powerful : 1  
language : 1  
widely : 1  
used : 1  

## Concepts Used

- Java File API
- HashMap
- String processing
